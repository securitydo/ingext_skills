# Release notes

Change history for the skills in this repo, for ease of tracking.

## Convention

- Release notes are organized **by date**. One file per release date, named
  `YYYY-MM-DD.md` (e.g. `2026-06-18.md`).
- When you change a skill, add (or append to) the file for the current date —
  note the skill name and what changed. Each skill's `SKILL.md` frontmatter also
  carries a `version:` field; mention the new version in the entry if you bumped it.
- List the newest dates first in the history table below.
